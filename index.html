Características Principales

🎯

Sintaxis Declarativa

Estructura clara inspirada en YAML que hace el código más legible y fácil de mantener para desarrolladores de todos los niveles.

⚡

Transpilación Automática

Se convierte a Python válido en tiempo real, manteniendo toda la potencia del ecosistema Python sin comprometer el rendimiento.

📚

Compatible con Python

Usa cualquier librería del ecosistema Python. Acceso completo a NumPy, Pandas, Django y miles de paquetes más.

🔧

Zero Dependencias

Solo necesita Python estándar (3.6+). Sin instalaciones complejas ni configuraciones adicionales. Funciona out-of-the-box.

🎨

VS Code Integration

Extensión incluida con resaltado de sintaxis, autocompletado y snippets para una experiencia de desarrollo optimizada.

📖

Fácil de Aprender

Sintaxis intuitiva perfecta para principiantes, pero lo suficientemente potente para desarrolladores experimentados.

Ejemplos de Código

🐍 Sintaxis PyML

# Variables y funciones
name: "PyML"
version: 1.0

# Estructura de datos
person:
  name: "Ana"
  age: 25
  skills:
    - "Python"
    - "YAML"

# Función
function greet define name:
  print: "¡Hola {name}!"

# Bucle
for skill in person.skills:
  print: "Skill: {skill}"

greet: person.name

⚡ Python Generado

# Variables y funciones
name = "PyML"
version = 1.0

# Estructura de datos
person = {
    "name": "Ana",
    "age": 25,
    "skills": ["Python", "YAML"]
}

# Función
def greet(name):
    print(f"¡Hola {name}!")

# Bucle
for skill in person["skills"]:
    print(f"Skill: {skill}")

greet(person["name"])
